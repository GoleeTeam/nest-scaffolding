## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run tests

# e2e tests
$ pnpm run tests:e2e

# tests coverage
$ pnpm run tests:cov
```
