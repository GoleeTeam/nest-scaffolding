## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit and component tests
$ pnpm run tests

# api tests
$ pnpm run tests:api

# integration tests
$ pnpm run tests:integration

# tests coverage
$ pnpm run tests:cov
```
